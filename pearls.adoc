[preface]
== 前言

网上冲浪，偶尔会遇到非常好的资料。但看完，经常就找不到了。另外，有些非常系统的资料，但是没有 GitHub 仓库，不能 Fork。整理一个文档，收集一些比较适合学生学习的资料。文档由此而来。

命名借鉴《Programming Pearls》（中文名：《编程珠玑》），取“珠玑”二字，表示弥足珍贵之意。

== 优质资料

. https://roadmap.sh/[Developer Roadmaps^] -- 这个 Roadmap 太全面了，不要被其吓到，慢慢来，会逐步跟上的。
. https://docs.microsoft.com/en-us/learn/paths/web-development-101/[Web development for beginners - Learn | Microsoft Docs^]
. https://exploringjs.com/deep-js/toc.html[Deep JavaScript^] -- 一本免费阅读的英文电子书，介绍 JavaScript 语言的底层机制和设计思想。
. https://www.pdai.tech/md/outline/x-outline.html[Java 全栈知识体系^]
. https://snailclimb.gitee.io/javaguide/[JavaGuide^]
. https://github.com/jwasham/coding-interview-university[A complete computer science study plan to become a software engineer.^]
