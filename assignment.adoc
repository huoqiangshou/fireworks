== 基于 TinyMVC + FreeMarker 实现的商城

基于上面的项目，自己实现一个 MVC 框架：TinyMVC。

将 TinyMVC 抽取出来，做个一个独立的小项目，可以在其他地方复用。

开发框架，就需要重点考虑这个问题：怎么加载未来的类？说的更通俗易懂些，就是如何加载使用这个框架的业务代码。这里就需要学习反射、注解、类加载等知识。
